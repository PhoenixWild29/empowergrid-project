# EmpowerGrid Critical User Journeys

## Investor Journey

1. **Discovery**  
   - Browse curated hero projects and recommended groups on the marketplace.  
   - Apply multi-dimensional filters (technology, impact focus, geography, risk band).  
   - Save searches and bookmark promising projects for comparison.  

2. **Due Diligence**  
   - Review project detail hero with impact summaries and validator credibility.  
   - Inspect milestone escrow tracker, funding status, and verifier notes.  
   - Explore developer profile, community updates, and third-party validation artifacts.  

3. **Wallet Connection**  
   - Walk through wallet primer (Phantom/Solflare) and safety check-list.  
   - Connect wallet and run readiness check (SOL balance, network status, signer permissions).  

4. **Investment Transaction**  
   - Enter amount, preview gasless escrow deposit, and confirm milestone-based release schedule.  
   - Receive real-time confirmation, escrow receipt, and portfolio allocation update.  

5. **Portfolio Tracking & Milestone Updates**  
   - Monitor growth analytics, impact metrics, and personalized insights.  
   - Receive push and email notifications for validator approvals, delays, or disputes.  
   - Reinvest matured capital directly from the dashboard or auto-roll into curated lists.  

## Developer Journey

1. **Onboarding**  
   - Complete guided profile setup (team credentials, governance model, compliance docs).  
   - Select mentoring track (first-time developer vs returning) with contextual resources.  

2. **Project Proposal Wizard**  
   - Step-through wizard: concept, technical specs, funding structure, community impact.  
   - Inline validation for KPIs (capex/opex mix, energy yield, projected ROI).  
   - Real-time preview of investor pitch layout.  

3. **Milestone Definition**  
   - Break project into validator-ready tranches with budget, evidence requirements, and oracle hooks.  
   - Assign validator types (community, engineering, environmental) and submission templates.  

4. **Fundraising Phase**  
   - Publish project to marketplace with status badges and funding bar.  
   - Access messaging prompts and update templates for investor trust building.  

5. **Milestone Verification & Fund Release**  
   - Submit documentation via wizard, attach oracle feeds, and request validator review.  
   - Track verification SLA, resolve feedback, and signal readiness for tranche release.  
   - Receive disbursement confirmation and automatically update public progress log.  

6. **Post-Launch Operations**  
   - Report ongoing performance metrics and impact insights.  
   - Issue community rewards, manage governance proposals, and plan scaling rounds.
